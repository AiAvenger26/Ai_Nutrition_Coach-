# AI Nutrition Coach

## Run Instructions

1. Open terminal in project folder:
```
cd C:/Users/mdtay/Desktop/AI_Nutrition_Coach
```

2. Install dependencies:
```
pip install -r requirements.txt
```

3. Run the app:
```
python app.py
```

4. Open browser to `http://127.0.0.1:5000`
